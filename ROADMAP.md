# Roadmap

## Near term
- Improve API documentation with more examples and edge cases.
- Add optional export to JSON/CSV.
- Add basic auth for API endpoints (optional).

## Mid term
- Add pluggable scanners and banner payload packs.
- Provide a lightweight UI for scan management.
- Add structured logging and metrics.

## Long term
- Distributed scanning (multi-node).
- Role-based access control for multi-user setups.
- Integrations with common security tooling.
